# Marp Slides for Professional Presentations

Create investor-ready presentations using Markdown-based Marp, exportable to PDF, HTML, or PowerPoint.

## Primary Use Cases
- LP quarterly update decks
- IC acquisition presentation slides
- Converting a whitepaper or memo into a slide deck
- Fund-raising decks
- Market analysis presentations

## Core Syntax
Slides are separated by `---`. Add speaker notes with `<!-- note: ... -->`.

## Export Options
| Format | Best For |
|--------|----------|
| PDF | Distribution |
| PowerPoint (.pptx) | Editable versions |
| HTML | Web viewing |

Run with watch mode for live preview: `marp --watch slides.md`

## Recommended Workflow
1. Draft narrative content
2. Structure into slides (one concept per slide)
3. Integrate data visualizations
4. Apply branding (custom theme in frontmatter)
5. Export and refine in PowerPoint as needed

## Slide Count Guidelines
- Quarterly LP updates: 8–12 slides
- Fund-raising decks: 15–20 slides
- IC acquisition memos: 10–15 slides

## Key Principles
- One concept per slide
- Numbers must be properly sourced
- Lead with the most important metric
- Use consistent color palette (2–3 colors max)
