# Data: Analyze

Analyze structured datasets — identify trends, outliers, and insights with clear narrative summaries.

## Key Capabilities
Portfolio performance, rent trends, expense benchmarking, tenant payment behavior, and CapEx spend analysis.

## Six-Step Protocol

1. **Understand the Data** — Clarify column definitions, time periods, granularity, and data quality issues
2. **Descriptive Summary** — Key metrics in tabular format (totals, averages, top/bottom performers)
3. **Trend Analysis** — Period-over-period changes and inflection points
4. **Outlier Identification** — Flag records exceeding 2 standard deviations or 20% variance thresholds
5. **Benchmarking** — Compare against budget, prior periods, portfolio averages, and market data
6. **Narrative Summary** — Headline finding, supporting bullets, anomalies, and implications

## Common Industrial RE Applications
- Rent roll analysis (weighted average rents, lease expirations, tenant concentration)
- Expense analysis (OpEx per SF, expense ratios, category breakdowns)
- Collections analysis (collection rates, receivables aging, payment patterns)

## Output Structure
Data summary → findings → outlier tables → trend analysis → actionable recommendations
