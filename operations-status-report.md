# Operations: Status Report

Generate clear status reports on asset, project, or fund progress for internal and external stakeholders in industrial real estate.

## Key Templates

**Weekly Internal Format**
- Highlights
- Leasing metrics
- Operations data
- Financial snapshot
- Upcoming tasks
- Escalations

**Monthly LP Asset Report**
- Property overview
- Financial performance vs. budget
- Leasing summary
- Capital project tracking
- Market analysis
- Forward-looking commentary

## Guiding Principles
Use numbers; avoid vague language ("good" → "$X, up 5% from budget"). Tone differs by audience:
- Internal: direct, flag problems clearly
- External LP: professional, contextualized

## Use Cases
- Weekly team updates
- Monthly investor reports
- Construction progress tracking
- Leasing activity summaries
- Portfolio-level fund status reviews
