# Sales: Pipeline Review Framework

Analyze industrial real estate pipelines for acquisition and leasing workflows.

## Acquisition Pipeline Stages
| Stage | Probability |
|-------|-------------|
| Sourced | 5–15% |
| Initial Review | 20–30% |
| LOI | 40–60% |
| Due Diligence | 70–80% |
| Under Contract | 85–90% |
| Closing | 90–99% |

## Leasing Pipeline
Track tenant prospects from initial inquiry through lease execution. Measure occupancy velocity by property.

## Critical Metrics
- Total pipeline value (probability-weighted)
- Stage distribution mapping
- Conversion rates to identify bottlenecks

## Pipeline Health
- 🟢 Green — diversified stage distribution
- 🟡 Yellow — concentration risk
- 🔴 Red — insufficient activity or stalled deals

## Weekly Report Structure
1. Active deal count
2. Weighted pipeline valuation
3. Near-term closings (90-day window)
4. Advancing deals
5. At-risk items requiring intervention
6. Decisions pending leadership approval (with deadlines)

**Users:** Asset Managers, Fund Managers | **Supports:** capital deployment tracking, deal prioritization, LP communication
