# Productivity: Task Management

Organize daily tasks and priorities across asset management, finance, deals, and LP relations.

## Priority Matrix (Eisenhower)
| Quadrant | Criteria | Action |
|----------|----------|--------|
| DO NOW | Deadline today, critical path | Execute immediately |
| SCHEDULE | High-value, no immediate deadline | Block time |
| DELEGATE | Tasks others can handle | Assign with due date |
| ELIMINATE | Low-value, non-urgent | Drop it |
