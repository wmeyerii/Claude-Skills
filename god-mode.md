# God Mode: Extended Autonomous Operation

Activates extended autonomous operation for complex multi-step industrial real estate workflows — Claude plans, executes, and iterates with minimal interruptions.

## Trigger
`/god-mode [task description]`

## What This Mode Enables
- Plan complete workflows before execution
- Progress through steps independently without constant confirmation
- Make reasoned judgments while documenting decisions
- Deliver finished outputs with decision summaries
- Only escalate genuine blockers (missing data, unclear instructions, irreversible actions)

## Three-Phase Structure

**Phase 1 — Planning**
Show: estimated steps, outputs, autonomous decisions to be made, and potential blockers.

**Phase 2 — Execution**
Process sequentially with progress markers: `[Step 3/12 complete]`

**Phase 3 — Delivery**
Structured summary including: outputs produced, autonomous decisions made, exceptions requiring review, and next steps.

## Built-In Safeguards
Human confirmation is always required for:
- External communications (emails, LP reports)
- Irreversible financial transactions
- Decisions exceeding $50K financial impact
- Overriding existing records

## Ideal Applications
Tasks with clear input data, well-defined outputs, repetitive sequential workflows, and multi-asset analysis across industrial real estate portfolios.
