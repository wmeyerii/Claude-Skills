# Data: Build Dashboard

Create KPI dashboards tracking occupancy, NOI, IRR, and rent collections for industrial real estate portfolios.

## Design Principles
- Lead with the most important metric
- Show trend, not just a number
- Audience-first: external LP reporting differs from internal management scorecards

## Dashboard Types

**Portfolio Management Dashboard** (monthly, internal)
- Occupancy by property
- NOI actual vs. budget
- Collections rate
- CapEx spend vs. plan

**LP Quarterly Report Dashboard** (external)
- Fund-level returns (IRR, equity multiple)
- Portfolio occupancy and NOI
- Capital deployed vs. committed

**Rent Collection Dashboard**
- Collection rate by property
- Delinquency aging buckets
- Outstanding balances

## Build Steps
1. Identify key metrics for the audience
2. Calculate period comparisons (MoM, YoY, vs. budget)
3. Flag variances against budget
4. Apply traffic light status indicators (green/yellow/red)
5. Deliver in clean tabular format with narrative summary

## Output Format
Structured table with actual vs. budget, prior period comparison, and health indicators suitable for executive consumption.
