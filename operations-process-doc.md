# Operations: Process Documentation (SOP)

Structured framework for creating Standard Operating Procedures in industrial real estate settings.

## Core Components

**Essential sections:**
- Purpose statement (1–2 sentences explaining significance)
- Defined scope with inclusions and exclusions
- Role and responsibility assignments
- Sequential process steps with time estimates
- Decision matrices for edge cases
- Escalation protocols

## Writing Approach
Use active voice and action verbs ("Submit," "Review," "Notify"). Name roles rather than individuals so documentation remains stable across personnel changes. Each step = one action; avoid combining multiple steps.

## Common SOP Applications
- Rent collection and tenant payment processing
- Vendor invoice processing and AP workflow
- Lease renewal process
- Property inspections
- CAM reconciliation
- Maintenance request handling
- Move-in / move-out procedures

## Output Format
- Title, version, owner, effective date
- Purpose and scope
- Roles and responsibilities table
- Step-by-step procedure (numbered, with role and estimated time)
- Decision tree for exceptions
- Escalation path
- Related documents / forms
