# Data: Create Visualizations

Create charts and data visualizations for industrial real estate reports, LP decks, and presentations.

## Primary Use Cases
- LP quarterly report charts (NOI, occupancy, returns)
- IC presentation data visualization
- Market analysis charts (rent trends, vacancy, supply)
- CapEx tracking waterfalls
- Portfolio composition charts
- Fund performance vs. benchmark comparisons

## Chart Type Reference
| Chart Type | Best For |
|------------|----------|
| Bar chart | Category comparisons |
| Line chart | Temporal trends |
| Stacked bar | Composition evolution |
| Waterfall | Build-ups / breakdowns |
| Pie chart | Portfolio composition |
| Scatter plot | Correlations |
| Heat map | Performance matrices |

## Technical Standards
- Limit color palettes to 2–3 colors maximum
- Always include axis labels, units, and data sources
- Avoid 3D charts (perceptual distortion)
- Mermaid chart templates available for common scenarios

## Output Formats
- Mermaid diagrams (for Claude Code rendering)
- Formatted data tables (for spreadsheet use)
- ASCII text-based representations
- All accompanied by written description and key insights
