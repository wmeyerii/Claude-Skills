# Finance: Variance Analysis Framework

Structured approach for analyzing budget versus actual performance in industrial real estate.

## Four-Step Framework

**1. Data Collection**
Gather actual P&L statements (current month and YTD), budget comparators, prior month results, and documentation of one-time items.

**2. Variance Calculation**
Build a table showing budget, actual, dollar variance, percentage variance, and whether each line item is favorable (revenue exceeding budget or expenses below budget).

**3. Driver Identification**
Focus on material variances exceeding $5,000 or 7% (scale threshold to fund size). Revenue drivers include occupancy changes and lease adjustments. Expense drivers encompass timing differences and vendor pricing shifts.

**4. Commentary Format**
- Account name
- Variance amount
- Specific drivers
- YTD impact
- Forecast trajectory
- Required actions

## Materiality Thresholds
Scale with portfolio size — ranging from $2,500 for smaller funds to $10,000 for larger portfolios.

## Output
Variance table + bullet-point analysis of material variances + summary language suitable for investor or management reporting.
