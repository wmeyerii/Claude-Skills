# Finance: Journal Entry Preparation

Prepare, review, and post journal entries including accruals, reclassifications, and adjusting entries.

## Use Cases
- Month-end accruals (unpaid invoices, utilities, management fees)
- Depreciation and amortization entries
- Prepaid expense amortization
- Reclassification of miscoded transactions
- Intercompany eliminations
- Error correction entries
- Deferred revenue recognition

## Required Fields for Every JE
| Field | Description |
|-------|-------------|
| Date | Period end date or transaction date |
| JE Reference | Unique ID (e.g., JE-2025-03-001) |
| Description | Clear, specific narrative |
| Debit Account | Account code + name |
| Credit Account | Account code + name |
| Amount | Dollar amount |
| Preparer | Name |
| Approver | Name |
| Support | Attached document reference |

## Standard JE Format
```
JE Reference: [ID]
Date: [YYYY-MM-DD]
Entity: [LLC / Fund Name]
Description: [Clear explanation of what and why]

  DEBIT   [Account Code] [Account Name]     $[Amount]
  CREDIT  [Account Code] [Account Name]     $[Amount]

Support: [Invoice #, contract, calculation]
Preparer: [Name] | Approver: [Name]
```

## Common JE Types

### Accrued Expenses
Debit: Expense account | Credit: Accrued Liabilities (2100)

### Prepaid Expense Amortization
Debit: Insurance Expense | Credit: Prepaid Insurance

### Depreciation
Debit: Depreciation Expense | Credit: Accumulated Depreciation

### Reclassification
Debit: Correct account | Credit: Original (incorrect) account

## Review Checklist
- [ ] Debits = Credits
- [ ] Entity is correct
- [ ] Period is correct and open
- [ ] Support document attached or referenced
- [ ] Description is clear enough for an auditor
- [ ] Reversal date set (if applicable)
